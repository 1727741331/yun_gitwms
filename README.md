# 吉特仓储管理系统-云服务版本
该版本将Web显示端与API端做了分离,新增了多仓库，多企业，采购销售，以及财务部分模块。 便于二次开发，可以快速应对一部分中小型客户需求，2018年个人所开发的所有项目均在此基础上深化而来。

# 主要功能模块
### 系统设置

(1)员工管理;
(2)角色管理;
(3)部门管理;
(4)资源管理;
(5)权限分配;
(6)标识符管理;

### 基本资料

(2) 库位管理;

(4) 供应商管理;
(5) 客户管理;
(6) 计量单位;
(7) 产品类别;
(8) 产品管理;
(9) 承运商管理;


### 仓库作业

(4) 移库管理;
(5) 盘点管理;
